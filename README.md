# Safety-Aware Navigation for Apple Maps

This is a code bundle for Safety-Aware Navigation for Apple Maps. The original project is available at https://www.figma.com/design/ezWEhTRRiIsgBXHDZyQNkI/Safety-Aware-Navigation-for-Apple-Maps.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

## Deploying to GitHub Pages

This site is automatically deployed to GitHub Pages using GitHub Actions whenever you push to the main branch.

Visit: https://gsanchezgomez.github.io/Safety-Aware-Navigation-Case-Study/
